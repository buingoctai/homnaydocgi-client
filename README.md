# homnaydocgi-fe

## Node version: 14.4.0

## Install packages:

```
yarn
```
or
```
npm install
```
## Run app:
```
yarn dev
```

## Formating Code:
```
yarn format
```
## Heorku:

- Logs: 
```
heroku logs --tail --app homnaydocgi-client
```
- Restart app: heroku restart -a homnaydocgi-client
